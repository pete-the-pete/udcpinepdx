# Security policy

This is a personal learning project that drives a wood-fired pizza oven door.
It is not production software, has no support SLA, and carries no warranty
(see `LICENSE`).

## Reporting a vulnerability

If you find a security issue — for example a secret accidentally committed to
history, a credential leak, or a remote-exploitable bug in any code that ends
up on the Raspberry Pi — please report it **privately** via GitHub's private
vulnerability reporting on this repository's **Security** tab rather than
opening a public issue.

A response is best-effort. Fixes will be rolled into the normal plan-driven
development cycle.
